# 신발빨래방 로컬 POS (v1)

Windows PC 1대에서 동작하는 로컬 POS입니다.  
SQLite 단일 파일 저장, 자동 백업, CSV 내보내기를 지원합니다.

## 실행 방법

```bash
cd ShoesFosSystem
dotnet run
```

또는 Visual Studio에서 `ShoesFosSystem.sln`을 열고 F5로 실행합니다.

## 주요 기능

- **메인(오늘)**  
  오늘 날짜, 날짜 선택 필터, 새 주문 / 가격표 설정 / 매출·리포트 / 백업 폴더 열기, 오늘 주문 리스트, 주문 수·총 매출·취소 수 요약

- **새 주문 / 수정**  
  접수번호 자동(매일 1번부터), 고객명·전화, 품목 여러 줄(신발 종류·수량·단가·추가금·메모), 라인합계·주문 합계 자동 계산, 저장 / 저장+완료 / 취소처리 / 닫기, 추가금 프리셋(+3000/+5000/+10000)

- **가격표 설정**  
  신발 종류 목록, 추가·수정·판매중지(비활성)

- **매출/리포트**  
  기간(시작일~종료일) 선택, 기간 총매출(취소 제외), 일자별 매출, 신발종류별 매출, CSV 내보내기(orders, order_items)

- **백업**  
  앱 시작 시·종료 시 `backup/pos_yyyyMMdd_HHmm.db` 자동 생성, 30개 초과 시 오래된 파일 삭제

## 데이터

- DB 파일: 실행 파일과 같은 폴더의 `pos.db`
- 백업: `backup/` 폴더
- CSV 내보내기: `export/` 폴더

## UI

- 60세 이상 사용자를 고려해 기본 폰트 크기(12pt), 버튼·라벨 13~14pt 적용
- 맑은 고딕, 단순하고 읽기 쉬운 구성

## 기술 스택

- .NET 8, Windows Forms, Microsoft.Data.Sqlite, SQLite (WAL 모드)
